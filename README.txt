How to build:
1. Create GitHub repo and upload pubspec.yaml and lib/main.dart and android/ folder (preserve folder structure).
2. Add codemagic.yaml to repo root (optional but recommended).
3. Use Codemagic.io connected to GitHub to build Android release build.
4. Download app-release.apk from Artifacts and install on device.
